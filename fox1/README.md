# fox1
The MDM MDM-1 Fox is a Polish, composite mid-wing two-seater aerobatic glider with fixed undercarriage and conventional tail unit. Things you have to know : - The aerobatic sequence displayed on the panel is quite difficult to read. If you want to see it better, jou just have to click on it, and it will be displayed in a way that you see it on your side, closer to you. - The aerobatic version max g-load is +9/-6, but fox the utility version it's only +5.3/-2.7. Don't forget it ! 

Made by Krzysztof Kaniewski. 
3D Design: Sim-Air (http://xpfr.org/)
